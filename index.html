HTML

HTML ou Linguagem de Marcação de Hipertexto é uma linguagem usada na criação de sites, que se utiliza de tags para estruturar páginas da web, aqui estão as principais tags organizadas em um Guia de Referências HTML.

"O HTML é o ponto focal de tudo o que se faz em desenvolvimento web. Não importa o que você faça, o HTML é o que há de mais importante na web." (EIS, Diego, p.11)

<h1> a <h6>

As tags <h1> a <h6> são usadas para definir cabeçalhos HTML.

  1. <h1> define o cabeçalho mais importante
  2. <h2>
  3. <h3>
  4. <h4>
  5. <h5>
  6. <h6> define o título menos importante.
